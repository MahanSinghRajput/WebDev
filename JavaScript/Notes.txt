Dynamically typed language. (means datatype not required)
primitive datatypes - string, boolean, undefined, int, bigInt, Number, Null, Symbol
non-primitive - object, array, function (compkex, mutable)

var ->
Scope: Function scoped
Re-declaration: allowed within the same scope
Reassignment: allowed
Hoisting: variables are hosted and initialised with undefined

let ->
Scope: Block-scoped({})
Re-declaration: Not allowed in the same scope
Reassignment: allowed
Hoisting: Hoisted but not initialised (Temporal dead zone)

const ->
Scope: Block-scoped({})
Re-declaration: Not allowed in the same scope
Reassignment: Not allowed
Hoisting: Hoisted but not initialised (Temporal dead zone)

# == vs ===
var v = 10
var r = "10"
console.log(v == r)  // true
console.log(v === r) // false
